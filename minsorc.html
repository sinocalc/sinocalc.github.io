Buff/Debuff Calculator
  1. Fill in the P.Atk, M.Atk, P.Def, M.Def values.
  2. Select the weapon type for each slot. If slot is empty, leave it on none.
  3. Select the number of weapon targets.
  4. Select the attributes the weapon buffs/debuffs.
  5. Select the skill tier for each weapon and fill in the skill level.
  6. If the weapon has Support Boon, select the tier and fill in the skill level.
  7. Grid and weapon names are optional, use them at your convinience.
  8. Click the calculate button once completed.
Grid Comparator
  • Click compare grids to open second grid input.
  • Fill out, copy, or import a second grid.
  • Click the compare button once both grids are completed.
Analysis
  • By default average weapon buffs and buffs from one grid rotation is calculated.
  • SB average is calculated with proc rate and skill level.
  • AoE average is calculated with range of targets.
  • Alternatively, specific SB procs can be selected, instead of average.
  • The comparator will find the grid with greater total buffs from one rotation.
  • Use the information to refine your grid and determine the best setup.
  • If you can't find a weapon skill, it's mods are likely missing from the references used.
