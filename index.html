How-to
  1. Fill in the P.Atk and M.Atk values.
  2. Select the corresponding weapon type for each slot. If slot is empty, leave it on none.
  3. Select the corresponding skill tier for each weapon.
  4. Fill in the skill level of each weapon.
  5. If the weapon has Dauntless Courage, select the tier and fill in the skill level.
  6. Weapon names are optional and only help keep track of the weapon.
  7. You can copy the grid over to make adjustments or create the second grid from scratch.
  8. Click the compare button once both grids are filled out correctly.
Features
  • Grids can be copied to one another, overwriting existing information.
  • Click export to download grids as a json file.
  • Click import and select an exported json file to import that grid.
Results
  • Almost all modifiers are applied after the atk - def is calculated.
  • If the grid atk values are different, the damage differential varies according to defense.
  • Example: (11 atk - 0 def) / (10 atk - 0 def) is 110% but (11 atk - 5 def) / (10 atk - 5 def) is 120%
  • After clicking compare, use the sliders to test different defense values.
  • You can dynamically view the damage differential between the grids as the defense value changes.
  • Use this as reference on how the grids compare at different defense values.
  • Depending on the grids, there may be breakpoints where the grids reverse in strength.
  • This is only a simple simulation, ultimately it's up to you to figure out how to best use this information.
