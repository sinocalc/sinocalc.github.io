How-to
  1. Fill in the P.Atk and M.Atk values.
  2. Select the corresponding weapon type for each slot. If slot is empty, leave it on none.
  3. Select the corresponding skill tier for each weapon and fill in the skill level.
  4. If the weapon has Dauntless Courage, select the tier and fill in the skill level.
  5. Grid and weapon names are optional, use them at your convinience.
  6. Copy the grid over to make adjustments or create the second grid from scratch.
  7. Click the compare button once both grids are filled out correctly.
Features
  • Grids can be copied to one another, overwriting existing information.
  • Click export to download the selected as a json file.
  • Click import and select an exported json file to import that grid.
Results
  • The grid damage is an estimate of the damage of a full rotation.
  • Almost all modifiers are applied after the atk - def is calculated, except combo.
  • If atk values differ, the damage differential varies based on defense and combo.
  • Ex: If combo = 0 and mods were 1, comparing 11 and 10 atk vs 0 and 5 def.
    (11 atk - 0 def) / (10 atk - 0 def) = 110%
    (11 atk - 5 def) / (10 atk - 5 def) = 120%
  • When comparing, use the sliders to test different defense and combo values.
  • Use this as reference on how the grids compare at different values.
  • Depending on the grids, there may be breakpoints where the grids reverse in strength.
  • This is only a simple simulation, ultimately it's up to you to figure out how to best use this information.
